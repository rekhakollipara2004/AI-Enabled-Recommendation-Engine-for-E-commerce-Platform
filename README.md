# AI Enabled Recommendation Engine for an E-Commerce Platform

This project focuses on building an AI-enabled recommendation system for an e-commerce platform using Python and Streamlit.  
The system provides personalized product recommendations by applying multiple recommendation techniques such as rating-based, content-based, collaborative filtering, and hybrid approaches.

---

## Project Overview

Recommendation systems play a crucial role in modern e-commerce platforms by improving user experience and increasing customer engagement.  
This project demonstrates how different recommendation algorithms can be combined to suggest relevant products based on user behavior, ratings, and product similarity.

---

## Key Features

- Displays trending products for guest users
- Provides personalized recommendations for registered users
- Implements multiple recommendation strategies:
   1. Rating-Based Recommendation
        - This approach recommends products with the highest ratings and review counts. It is mainly used for guest users who do not have any interaction history.
  2. Content-Based Filtering
        - This method recommends products similar to the items previously rated or viewed by the user based on product attributes.
  3. Collaborative Filtering
        - This technique identifies users with similar preferences and recommends products liked by those users.
  4.  Hybrid Recommendation
        - The hybrid approach combines content-based and collaborative filtering techniques to improve recommendation accuracy and relevance.

- Identifies users with similar interests
- Shows recent user activity
- Product search functionality
- Interactive user interface developed using Streamlit
- Custom dark-themed user interface with responsive design
